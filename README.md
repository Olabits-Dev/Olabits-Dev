## Featured Projects

### [GMT Homes Real Estate Platform](https://github.com/Olabits-Dev/GMT-Homes-Real_Estate)
A polished real estate platform built with Next.js, featuring property browsing, role-based authentication, dashboards, listing submission, moderation workflows, booking flows, PostgreSQL integration, maps, and cloud-ready uploads.

### [AI Job Hunter Bot](https://github.com/Olabits-Dev/job-alert-bot)
An automation tool powered by GitHub Actions and Resend that scans job boards, filters and ranks roles, supports multiple profiles, discovers recruiter emails, and sends daily job reports.

### [MT5 SaaS Automated Dashboard](https://github.com/Olabits-Dev/Mt5-SaaS-Automated-Dashboard)
A dashboard built around MT5 SaaS operations, client visibility, and trading workflow management.

### [MT5 Multi-Client SaaS Intraday Bot](https://github.com/Olabits-Dev/MT5-Saas-MultiClient-Intraday-Bot)
A multi-client MT5 trading system with encrypted client storage, VPS-ready deployment, Telegram alerts, and structured intraday execution logic.

### [Expense Tracker React App](https://github.com/Olabits-Dev/Expensetracker_refilne)
A React + Vite expense tracker with add, edit, delete, category filtering, monthly summaries, and local storage persistence.

### [Car Rental Platform](https://github.com/Olabits-Dev/Car-rental-platform)
A Next.js-based car rental platform project focused on booking and service-oriented product design.
